## FILE PREPARATION

Each album ZIP structure:
```
Album_Name/
  WAV/
    01_Track_Name.wav
    ...
  FLAC/
    01_Track_Name.flac
    ...
  cover_art.jpg
  liner_notes.pdf
```

Convert: `ffmpeg -i input.mp3 -c:a flac output.flac`
